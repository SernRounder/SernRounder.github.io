解题思路

编码方式分析

编码流程如下图所示：

  1. 将ASCII表中每个字符对应数字减去31
  2. 在新表中找到待编码字符x对应的新编号x1
  3. 用新表长度减去x1，找到对应位置上的字符x2
  4. 在旧表中找到x2对应的编号x3，记录下来。

构造解码函数
